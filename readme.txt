== Description ==
Documentation theme for lolitaframework.com

For more information about Lolita please go to https://lolitaframework.com

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Lolita in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel.
